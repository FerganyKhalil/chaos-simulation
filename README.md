# Chaos Simulation - Logistic Map

This project simulates a nonlinear chaotic system using the logistic map equation.

## Key Concepts
- Nonlinear dynamics
- Chaos theory
- Sensitivity to initial conditions

## Applications
- Secure communication systems
- Noise modeling
- Signal dynamics analysis

## Equation
x(n+1) = r * x(n) * (1 - x(n))
